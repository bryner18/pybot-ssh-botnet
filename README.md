# PyBot SSH Botnet

This project was made by Bryner. Feel free to use and change whatever you want.

## Features

**Bot Collecting**

* SSH Brute Force Attack

**Five Attack Methods**

* Command Execution

* HTTP/HTTPS Attack

* PING Attack

* SYN Attack

* ACK Attack

**Bot Counting**

* Bots Alive Function

**File Upload**

* File Uploading via SSH

## Requirements

At least 1 **bot** to start the **Attack**
